# My Web Resume (HTML & CSS)

This is my very first Web Development project! I built my personal resume from scratch to practice front-end fundamentals, code structure, and styling.

## Objective
Create a digital version of my CV to showcase my skills, education, and my long-term career goal of working as a Software Engineer in Japan.

## Technologies Used
*   **HTML5:** Page structure, semantic tags, and lists.
*   **CSS3:** Styling, layout (`margin` and `padding` for centering), and hover animations.
*   **Git & GitHub:** Version control, terminal commands, and repository management.

## What I Learned
During this project, I was able to put into practice:
*   How to structure a webpage using tags like `<ul>`, `<li>`, `<h1>`, and `<div>`.
*   How to link an external CSS stylesheet to an HTML document.
*   How to center blocks of content on the screen using `margin: 0 auto`.
*   How to add interactive hover effects to links using `::after`.
*   How to solve Git merge conflicts when pushing code to GitHub.
